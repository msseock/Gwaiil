## 💡 프로젝트 동기

가볍고 재미있는 방식으로 작은 목표를 달성하면서 동기부여를 받을 수 있는 앱을 만들기 위해 시작했습니다.


## 📌 목표

- 개인 학습
- SwiftData를 활용한 로컬 저장 구현

## ⚙️ 개발 환경

- Xcode 16.2 / iOS 17.0
- Swift 5
- SwiftUI + SwiftData


## 🧩 구현 기능

- 21일 목표를 과일로, 목표에 따른 행동을 1개의 조각으로 기록
- 조각 개수가 늘어날수록 진화하는 과일
- 목표별로 하루에 한 개의 조각만 추가 가능하도록 제한


## 📚 배운 점 / 트러블슈팅

| 주제 | 요약 |
|------|------|
| SwiftData | SwiftData에서의 CRUD 구현 과정 |
